# 欧迪精英房间自动加入插件

## 功能说明

这是一个浏览器扩展插件,用于在访问 `http://www.alang.run` 网站时自动执行以下操作:

1. **自动搜索房间**: 搜索名称为 "欧迪精英" 的房间
2. **自动加入房间**: 如果找到该房间,自动加入并输入密码 `114514`
3. **自动创建房间**: 如果未找到该房间,自动创建一个名为 "欧迪精英" 的房间,密码为 `114514`

## 安装方法

### Chrome/Edge 浏览器

1. 下载或克隆此项目到本地
2. 打开浏览器,进入扩展管理页面:
   - Chrome: 访问 `chrome://extensions/`
   - Edge: 访问 `edge://extensions/`
3. 在右上角启用 "开发者模式"
4. 点击 "加载已解压的扩展程序"
5. 选择本项目的文件夹 `music_extension`
6. 扩展安装成功!

### Firefox 浏览器

1. 下载或克隆此项目到本地
2. 打开浏览器,访问 `about:debugging#/runtime/this-firefox`
3. 点击 "临时载入附加组件"
4. 选择本项目中的 `manifest.json` 文件
5. 扩展安装成功!

## 使用方法

1. 安装插件后,直接访问 `http://www.alang.run` 网站
2. 插件会在页面加载完成后自动执行搜索和加入/创建房间的操作
3. 可以在浏览器控制台中查看插件的运行日志(按 F12 打开开发者工具)

## 技术说明

- **manifest_version**: 3 (使用最新的 Manifest V3 标准)
- **运行时机**: 页面加载完成后执行(`document_idle`)
- **权限**: 仅需要 `activeTab` 权限,只在目标网站上运行

## 调试说明

如果插件未能正常工作,可以按照以下步骤调试:

1. 打开浏览器开发者工具(F12)
2. 切换到 "Console" (控制台) 标签
3. 查找以 `[欧迪精英插件]` 开头的日志信息
4. 根据日志信息判断插件的执行状态

## 注意事项

- 插件会在页面加载后等待 2 秒,以确保动态内容加载完成
- 如果网站的页面结构发生变化,可能需要更新 `content.js` 中的选择器
- 请确保访问的是 `http://www.alang.run` 或 `https://www.alang.run`,插件只在这些域名下运行

## 常见问题

**Q: 插件没有自动执行怎么办?**  
A: 请检查插件是否已启用,并刷新页面重试。查看控制台日志确认插件是否加载。

**Q: 插件找不到房间或创建按钮怎么办?**  
A: 网站的页面结构可能与插件预期不同,需要根据实际页面结构调整 `content.js` 中的选择器。

**Q: 如何修改房间名称或密码?**  
A: 编辑 `content.js` 文件,修改开头的常量 `ROOM_NAME` 和 `ROOM_PASSWORD`。
